#  Calculator

A responsive calculator web app built using HTML, CSS, and JavaScript.

## 🚀 Features
- Basic arithmetic operations
- Dark mode toggle
- Calculation history (stored in LocalStorage)
- Sound effects on button click
- Input validation
- Responsive design for mobile & desktop

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 Project Structure
Calculator/
├── index.html  
├── script.js  
├── style.css  
├── utils.css  
├── images/  
├── sound-effect/  
└── Logo.png  

## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
